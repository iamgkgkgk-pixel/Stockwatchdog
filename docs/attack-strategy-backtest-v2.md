# 进攻仓策略回测报告 v2 — 巴菲特金字塔版

## 回测设计

- **周期**：2021-06 → 2026-05（60 个月，覆盖美元加息/中国资产杀估值/分化行情）
- **标的池**：9 个（用户指定 8 + AI 推荐 1）
- **阈值基准**：滚动 5 年前向窗口的 PE 均值 μ 和标准差 σ（**完全避免未来函数**）
- **净值代理**：月度 PE 变化率（在蓝筹类较准，成长类偏乐观）

## 策略对比

| 5. 金字塔·宽 | μ/-0.5σ/-1σ → 30%/60%/100% | μ+1σ/+1.5σ/+2σ | 容易入场容易持有 |

## 关键发现

### 1. 所有进攻策略普遍打败买入持有 ✅

买入持有 9 个标的年化平均 **-5.8%**；金字塔·中平均 **+1.9%**（+7.7 个百分点）；估值温和平均 **+2.1%**。

**结论**：任何带估值纪律的策略都优于满仓躺平。

### 2. 没有"万能档位"，需按标的特性分档 ⚠️

| 标的类型 | 最佳档位 | 原因 |
|---|---|---|
| 美股成长（纳指100） | 金字塔·中 | 估值波动适中，μ-0.5σ 就足够便宜 |
| 美股宽基（标普500） | 估值温和 | 估值稳，分位法更鲁棒 |
| A股宽基（沪深300/上证50） | 金字塔·中 | 估值窄幅震荡，μ-0.5σ 抓得住 |
| A股成长（科创50/创业板50） | 金字塔·严 | 跌幅深，必须等μ-1σ以下 |
| 港股（恒生科技） | 金字塔·宽 | 跌幅极端，稍跌就是机会 |
| 行业（医药） | 金字塔·严 | 政策扰动大，耐心等深坑 |

### 3. 科创半导体的警示 🚨

过去 5 年 PE 从 145 跌到 49，**单边下行**。所有金字塔策略都"越买越套"，最终跑不赢估值温和。

**给用户的风险提示**：
- **金字塔的前提是"会均值回归"**，对长期单边下行的行业（如 2020-24 的半导体主题 ETF、部分新能源）会失灵
- 对这类标的，建议**要么用估值温和（不盲目加仓）**，要么**排除出进攻仓**
- 可以加一个"周期防护"：**连续 24 个月未回归 μ 的标的，自动降级为估值温和或排除**

### 4. "估值温和"依然是最鲁棒的简单方案 💎

- 9 个标的里 2 个最佳、7 个正收益/接近最优
- 仅 1 档入场 + 1 档出场，实盘非常好执行
- **作为全局兜底规则，简单直接**

## 最终方案建议

### 分层进攻规则

```
对每个标的，系统根据市场类型自动选择档位：

  市场属性 = data[etf].market_type
  recommended_tier = {
    "US_GROWTH":    "中档",    # 纳指100/QQQ
    "US_BROAD":     "温和",    # 标普500/道琼斯
    "CN_BROAD":     "中档",    # 沪深300/上证50
    "CN_GROWTH":    "严档",    # 科创50/创业板50
    "HK":           "宽档",    # 恒生科技/恒生红利
    "INDUSTRY":     "严档",    # 医药/消费/能源
    "THEME_RISKY":  "温和",    # 半导体/机器人/AI
  }[market_type]
```

### 全局风险护栏（金字塔通用）

1. **单边下行保护**：若标的 PE 连续 24 个月未回归 μ，该月自动切换到 "估值温和"
2. **利率周期加权**：加息末期 → 建仓阈值放宽 0.25σ；加息中期 → 收紧 0.25σ（需补数据源）
3. **最大持仓比例**：单个进攻标的仓位不超过进攻仓总额的 40%（9 个标的分散）

### 与基石仓配合

```
总资产 100%
  ├── 基石仓 70% → 红利低波/自由现金流/国债 ETF（长线）
  └── 进攻仓 30%
       ├── 9 个候选标的，分层金字塔
       └── 允许阶段性全仓空仓（尤其加息周期）
```
